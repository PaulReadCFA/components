1 — Expandable Box

Needs JS

Dark blue header = closed. Bright blue = open. Add data-open to pre-expand. Math in tabs and equation boxes uses MathJax 3 (see end of page).

The time value of money (TVM) is the concept that money available today is worth more than the same amount in the future due to its potential earning capacity.

Add data-open to the .cfa-expand element and aria-expanded="true" to the button to start open.

▸ HTML snippet 
<div class="cfa-expand">
  <button type="button" class="cfa-expand__trigger" aria-expanded="false"
          aria-controls="panel-example">
    <span class="cfa-expand__icon" aria-hidden="true"></span>
    Your title here
  </button>
  <div class="cfa-expand__body" id="panel-example">
    <p>Hidden content.</p>
  </div>
</div>

<!-- Pre-open: add data-open + aria-expanded="true" -->
<div class="cfa-expand" data-open>
  <button type="button" class="cfa-expand__trigger" aria-expanded="true"
          aria-controls="panel-open">
    <span class="cfa-expand__icon" aria-hidden="true"></span>
    Open by default
  </button>
  <div class="cfa-expand__body" id="panel-open"><p>Visible from load.</p></div>
</div>

2 — Accordion

Needs JS

Wrap .cfa-expand items in .cfa-accordion. Open/Collapse all controls sit outside in .cfa-accordion__controls.

The TVM framework links present and future cash flows through discount rates. Key variables: PV, FV, r, and n.

FV = PV × (1 + r)n. For multiple cash flows, each is discounted to a common date and summed.

An annuity is a series of equal cash flows. A perpetuity is infinite: PV = C / r.

3 — Horizontal Tabs

Needs JS

Tabs are proper buttons on a grey track. Active = dark blue fill. Keyboard: ← → Home End. The Formula tab includes MathJax display and inline math (requires network for the CDN script).

Bond valuation is the process of determining the fair price of a bond — the present value of all expected cash flows.

\[ P = \frac{C}{1+r} + \frac{C}{(1+r)^2} + \cdots + \frac{C+F}{(1+r)^n} \]

Where \(C\) = coupon, \(F\) = face value, \(r\) = discount rate, \(n\) = periods.

3-year bond · 5% coupon · $1,000 face · 4% discount rate → P = $1,027.75

Try: 5-year, 6% coupon bond at a 7% discount rate. Answer: $958.99

    <button type="button" class="cfa-tabs__tab" role="tab"
            aria-selected="false" aria-controls="p2" id="t2">Tab 2</button>
  </div>
  <!-- First panel gets data-active -->

6 — CFA Program boxes

CSS only · Font Awesome

Colours align with the CFA Program curriculum box menu (e.g. cfa-curriculum-shaded-box, cfa-curriculum-exercise-box). Same centred-head shell as .cfa-box (section 7). For Example box, use fas fa-book-reader (or similar) rather than a magnifying glass so the icon is not mistaken for an expand control. Legacy modifiers --note, --key, --tip, --warning, --stop still work as aliases — prefer the curriculum names below for new markup.

Shaded box

Supplemental sidebar content — production cfa-curriculum-shaded-box.

Example box

Real-world example — cfa-curriculum-example-box.

Discussion

Open-ended prompt — same palette as the Discussion block in the curriculum menu.

Case study

Scenario shell — curriculum-case-study-box.

Exercise / knowledge check

Self-assessment, knowledge check, or question set — cfa-curriculum-exercise-box.

Learning outcomes

The candidate should be able to: … — cfa-curriculum-los-box. A plain cfa-callout with no modifier uses the same LOS shell.

Reference list

References block — cfa-curriculum-reflist.

LM overview / summary

Warm cream shell — cfa-curriculum-overview-box (deprecated tag in production; same look for module wrap-ups).

In this lesson, you will examine the assumptions of simple linear regression and learn how violations appear in residual plots. You will explore goodness-of-fit measures such as R², the F-statistic, and the standard error of the estimate.

Lesson Summary

In this lesson you examined how bond prices are determined by discounting future cash flows. You calculated prices for coupon bonds and zero-coupon bonds, and explored how changes in discount rates affect value.

Reflection

Before continuing, consider: if two bonds have identical coupons and face values but different maturities, how would you expect their prices to differ when interest rates change?

Worked Example

A 3-year bond has a face value of $1,000 and pays an annual coupon of 5%. The market discount rate is 4%. Calculate the fair value of this bond.

8 — Comparison Anchor

CSS only

Side-by-side concept cards to prevent conflation of related terms. Stacks on mobile.

Default (dark blue headers)

Linearity

Linearity concerns whether the relationship between X and Y follows a straight-line functional form.

Homoskedasticity

Homoskedasticity concerns whether the variance of residuals is constant across all observations.

A model may satisfy one assumption while violating the other.

Mixed header colours

Compare

Ordinary Annuity

Cash flows occur at the end of each period. Most loans and bonds follow this pattern.

Annuity Due

Cash flows occur at the beginning of each period. Leases are a common example.
